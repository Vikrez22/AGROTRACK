## Smart AgriTech Solution
# Ending Farmer-Herder Conflicts with Technology.
Agrotrack a smart agritech solution to reduce conflicts between farmers and herders, we use geofencing to define grazing and non-grazing zones. Our IOT device sends real time data to our dashboard and triggers and alert when an animals enter the non-grazing area. we also built an ai-powered chatbot to help educate both farmers and herders.
